
# Student Management System (Java)

A console-based Java application to manage student records.

## Features
- Add Student
- View All Students
- Update Student
- Delete Student
- Search Student by ID

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- Git & GitHub

## How to Run
1. Open in IntelliJ
2. Run StudentManagementSystem.java
3. Follow menu instructions
